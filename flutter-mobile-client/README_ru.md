# Мобильный Клиент на Flutter

Этот проект на Dart / Flutter демонстрирует интеграцию мобильного клиента с `nativebpm_client` SDK.

## Возможности
- **Лента задач**: Получение списка незавершенных ручных задач с помощью `client.tasks().list()`.
- **Захват и выполнение**: Использование API захвата (`client.tasks().claim()`) и завершения (`client.tasks().complete()`).
- **Мобильный интерфейс**: Удобная форма для вынесения решений операторами или менеджерами.

## Установка
Убедитесь, что у вас установлен Flutter SDK, и выполните:
```bash
flutter pub get
```

## Запуск
```bash
flutter run
```
