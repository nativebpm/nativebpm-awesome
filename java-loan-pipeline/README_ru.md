# Java Конвейер Кредитования

Это веб-приложение на Spring Boot, демонстрирующее запуск скоринга кредитных заявок с использованием Java SDK `nativebpm-java-client`.

## Возможности
- **Настройка ApiClient**: Конфигурация адреса сервера и токена доступа.
- **REST Контроллер**: Эндпоинт `POST /api/loans`, который отправляет запрос на старт инстанса процесса `loan-scoring-flow`.
- **Эмуляция сбоев**: Корректный ответ клиенту с переводом в локальный режим при недоступности внешнего движка.

## Установка
Сборка проекта с помощью Maven:
```bash
mvn clean compile
```

## Запуск
```bash
mvn spring-boot:run
```
Приложение запустится на порту `8080` (или другом свободном порту).
