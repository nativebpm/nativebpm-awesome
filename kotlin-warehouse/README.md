# Kotlin Warehouse Picker Example

This Kotlin JVM application demonstrates picking operations workflow task management using the Kotlin SDK client.

## Features
- **Task lists query**: Retrieves pending user tasks assigned to candidate groups.
- **Task claim flow**: Claims active tasks for individual picker assignees.
- **Payload completions**: Updates process state variables and completes tasks.

## Setup
Build the application using Gradle:
```bash
./gradlew build
```

## Run
```bash
./gradlew run
```
