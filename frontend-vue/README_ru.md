# Vue 3 Клиентский Интерфейс

SPA-приложение на Vue 3, Vite и Bootstrap, демонстрирующее кастомный реактивный рендеринг по схемам и интерактивную визуализацию BPMN-канала.

## Возможности
- **Кастомный парсер схем**: Динамическое сопоставление полей JSON Schema со стандартными элементами ввода HTML5.
- **Интерактивный трекер процессов**: Встраиваемый виджет выполнения процесса с подсветкой активных/завершенных нод.
- **Реактивность Vue 3**: Легковесное управление состоянием.

## Установка
```bash
npm install
```

## Запуск
```bash
npm run dev
```
Убедитесь, что `typescript-backend` запущен на порту 4000.
