# Kotlin Сборщик Заказов

Это приложение на Kotlin JVM, имитирующее терминал сборщика заказов на складе на основе Kotlin SDK.

## Возможности
- **Список задач**: Запрос невыполненных задач User Task по группам исполнителей.
- **Захват (Claim)**: Назначение активной задачи на конкретного сборщика.
- **Завершение (Complete)**: Обновление состава собранных товаров и закрытие задачи.

## Сборка
Сборка проекта с помощью Gradle:
```bash
./gradlew build
```

## Запуск
```bash
./gradlew run
```
