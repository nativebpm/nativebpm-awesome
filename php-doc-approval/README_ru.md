# PHP Согласование Документов

Этот скрипт под управлением Composer демонстрирует интеграцию PHP SDK для управления согласованием корпоративных документов.

## Возможности
- **Path-репозиторий**: Настройка автозагрузки локального PHP SDK (`sdk/php`) без публикации пакета в Packagist.
- **Интеграция с DefaultApi**: Запуск процесса `document-approval-flow` с передачей структурированных аргументов.
- **Обработка исключений**: Локальная эмуляция согласования в блоках `try-catch` при недоступности API-сервера.

## Установка
```bash
composer install
```

## Запуск
```bash
php index.php
```
