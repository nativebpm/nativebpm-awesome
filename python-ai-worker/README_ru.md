# Python AI-Воркер Агентов

Этот проект демонстрирует фоновый обработчик на Python, опрашивающий и выполняющий интеллектуальные задачи от NativeBPM.

## Возможности
- **Опрос задач (Polling)**: Длинные запросы (long-polling) для поиска активных задач, назначенных на `ai_agent`, через `client.tasks().list()`.
- **Имитация LLM-анализа**: Анализ входных данных заказа (сумма, имя клиента) с использованием текстовых промптов (эмуляция Gemini API).
- **Завершение через Fluent API**: Отправка вынесенного решения и примечаний обратно в движок.

## Установка
Создайте виртуальное окружение и установите пакеты:
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск
```bash
python main.py
```
