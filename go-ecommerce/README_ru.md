# Go E-Commerce Вебхук-Сервер

Этот сервер на базе фреймворка Gin демонстрирует запуск инстансов BPMN и проверку безопасности вебхуков от NativeBPM.

## Возможности
- **Процесс оформления заказа**: Запуск экземпляров процесса выполнения заказов через Go SDK.
- **Верификация HMAC-подписи**: Проверка входящих заголовков `X-NativeBPM-Signature` на секретный ключ для защиты от несанкционированных вызовов.
- **Работа без go.work**: Автономная сборка и скачивание зависимостей.

## Запуск
```bash
go run main.go
```
Сервер запустится на `http://localhost:8081`.
