# TypeScript Бэкенд Сервер

Этот сервер на Node.js/Express демонстрирует интеграцию пакета `@nativebpm/sdk` для управления бизнес-процессами.

## Возможности
- **Регистрация процессов**: Автоматически публикует воркфлоу `order-approval-flow` при старте.
- **Workflow-as-Code**: Программное описание сервисных и пользовательских задач, а также развилок (exclusive gateways).
- **Схемы форм**: Отдает динамические спецификации JSON Schema для выполнения ручных задач.
- **Визуализация процессов**: Предоставляет эндпоинты для получения интерактивного BPMN-виджета с помощью метода `client.instances().getVisualizationHTML()`.

## Установка
```bash
npm install
```

## Запуск
```bash
npm run dev
```
Сервер запустится на `http://localhost:4000`.
