# .NET API Заявок на Отпуск

Этот проект Web API на ASP.NET Core демонстрирует согласование отпусков сотрудников с использованием `NativeBPM.Client` SDK.

## Возможности
- **Маршрутизация**: Эндпоинт `POST /api/leaves` для старта инстансов процессов.
- **Типизированные переменные**: Пример сериализации бизнес-данных (`employeeName`, `daysCount`) в контекст процесса.
- **Отказоустойчивость**: Возврат локального статуса при недоступности движка.

## Сборка
Сборка проекта через интерфейс командной строки .NET:
```bash
dotnet build
```

## Запуск
```bash
dotnet run
```
API-сервер запустится на портах по умолчанию (`http://localhost:5000` / `https://localhost:5001`).
