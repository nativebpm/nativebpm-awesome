# React Клиентский Интерфейс

Приложение SPA на базе React, Vite и Bootstrap, демонстрирующее визуализацию процессов и динамические формы задач.

## Возможности
- **Формы JSON Schema**: Рендеринг форм с помощью библиотеки `@rjsf/core` по схемам, переданным из движка.
- **Список задач**: Отображение активных ручных задач, согласование и завершение шагов.
- **BPMN-визуализация**: Встраивание интерактивного SVG-фрейма схемы процесса с подсветкой токенов.

## Установка
```bash
npm install
```

## Запуск
```bash
npm run dev
```
Убедитесь, что `typescript-backend` запущен на порту 4000.
